🚀 Rocket vs UFO
Rocket vs UFO adalah game sederhana berbasis Python menggunakan library Pygame.
Pemain mengendalikan roket untuk menembak UFO yang muncul secara acak.
Project ini dibuat sebagai implementasi konsep Pemrograman Berbasis Objek (PBO).

Gameplay:
1. Player mengontrol roket (naik & turun)
2. Tekan spasi untuk menembak
3. UFO akan muncul dari kanan layar
4. Hancurkan UFO untuk bertahan

Cara Menjalankan:
1. Install Python
2. Install Pygame
3. pip install pygame
4. Jalankan program
5. python index.py
